## Overview

Seventh Tone News Reader is a cross-platform news reading application built for English learners. It aggregates curated English news from Sixth Tone and enhances the reading experience with AI-powered exercises, an integrated dictionary, sentence saving, vocabulary tracking, and rich reading analytics — all wrapped in a sleek, mobile-first interface deployable to both web and Android via Capacitor.

## Features

### 📰 News Browsing

- **Category-based navigation** — Browse news organized by topic categories with cover images and descriptions.
- **Infinite scroll** — Seamless pagination with IntersectionObserver; pull-to-refresh on mobile.
- **Full-text search** — Real-time keyword search with highlighted results and infinite loading.
- **Daily Tones** — A full-screen immersive carousel of daily curated picks with calendar-based date navigation.
- **List caching** — Previously loaded news lists are cached for instant back-navigation.

### 📖 Immersive Reading

- **Rich article rendering** — Full HTML content sanitized with DOMPurify, displayed with responsive typography.
- **Reading progress bar** — Visual scroll progress indicator at the top of the detail page.
- **Adjustable font size** — Three reading comfort levels: Small / Medium / Large.
- **Reading time tracking** — Automatic session-based reading duration stats, paused on tab switch.
- **Reading streak & analytics** — Dashboard with today/total article counts, reading time, calendar heatmap, and consecutive-day streak.

### 🔍 Vocabulary & Dictionary

- **Tap-to-lookup dictionary** — Double-tap any word for an instant bottom-sheet dictionary panel with phonetics, definitions, phrases, synonyms, and example sentences.
- **UK / US pronunciation** — In-dictionary audio playback for both accents.
- **Vocabulary notebook** — One-tap save/remove words with translations; manage in the dedicated Vocab tab.

### ✏️ Sentence Capture

- **Text selection & highlight** — Select any sentence in an article to save it to your sentence library with mark.js highlighting.
- **Category & notes** — Organize saved sentences into custom categories; attach personal thoughts.
- **Quote card generator** — Turn any saved sentence into a beautifully designed shareable image card via html2canvas.

### 🤖 AI-Powered Practice

- **Five exercise types** generated by LLM (SiliconFlow / Qwen):
  - Vocabulary fill-in-the-blank
  - Reading comprehension
  - English → Chinese translation
  - Chinese → English translation
  - Summary writing
- **Instant feedback** — Immediate right/wrong for multiple choice; AI scoring with feedback for translation and summary.
- **Card-style navigation** — Swipe through questions with a progress bar and answer sheet panel.

### 📱 Cross-Platform

- **Web + Android** — Single codebase deployed as a PWA-like web app and a native Android app via Capacitor.
- **Native integrations** — Status bar theming, safe-area handling, native share, save-to-gallery, filesystem access.
- **Data backup** — Export/import all local data (history, bookmarks, vocab, settings) as a JSON file.

### 🎨 Theming

- **Dark / Light mode** — System-aware theme with manual toggle; persisted across sessions.
- **Smooth animations** — Framer Motion page transitions and micro-interactions throughout.

## Quick Start

### Prerequisites

- **Node.js** ≥ 22.12.0
- **npm** (bundled with Node.js)
- **Android Studio** (optional, for Android builds)

### Installation

```bash
# Clone the repository
git clone https://github.com/<your-username>/seventh-tone-news-reader.git
cd seventh-tone-news-reader

# Install dependencies
npm install
```

### Development

```bash
# Start the development server (Express + Vite HMR)
npm run dev
# → http://localhost:3000
```

### Production Build

```bash
# Build for production
npm run build

# Preview production build
npm run preview
```

### Android Build

```bash
# Build web assets
npm run build

# Sync with Android project
npx cap sync android

# Open in Android Studio
npx cap open android
```

## Configuration

| Variable                       | Location         | Description                                     |
| ------------------------------ | ---------------- | ----------------------------------------------- |
| `VITE_SILICONFLOW_API_KEY`     | `.env`           | SiliconFlow API key for AI exercises            |
| `ACTIVE_MODEL`                 | `src/api/llm.ts` | Active LLM model (default: `Qwen2.5-7B`)        |
| `BASE_URL`                     | `src/api/api.ts` | Sixth Tone API base URL                         |
| `PORT`                         | `server.ts`      | Dev server port (default: `3000`)               |
